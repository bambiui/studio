# BambiUI Studio

Visual theme and component studio for BambiUI.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Deployment

This app is intended to be deployed as the Studio property at:

- https://studio.bambiui.com

The repository includes `vercel.json` for a Next.js deployment. Configure the custom domain `studio.bambiui.com` in the hosting provider/project settings and point DNS to the provider as required.

The public marketing/docs site links to this app from `https://bambiui.com`, and this app links back to:

- https://bambiui.com
- https://bambiui.com/docs/
