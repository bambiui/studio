# BambiUI Studio

BambiUI Studio is a visual theme and component playground for the BambiUI platform.

The current MVP lets you preview BambiUI components, override design tokens, generate a theme from a single base color, and export/import the resulting token map.

## Features

- Component explorer with search
- Selectable component canvas
- Draggable board canvas with persisted item positions
- Board item resize, duplicate and layer controls
- Board zoom and snap-to-grid controls
- Board layout import/export
- Keyboard movement for board items
- All/selected canvas view modes
- Light and dark preview modes
- Live token overrides through CSS variables
- Token controls for color, radius and opacity values
- Token search and token group filters
- Light and dark theme generation from a single base color
- Built-in theme color presets
- Contrast report
- CSS, JSON and BambiUI preset export with token summary
- CSS/JSON/preset import
- Local storage persistence

## Available components in the canvas

- Button
- Card
- Input

## Development

Install dependencies and run the app:

```/dev/null/commands.sh#L1-2
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Quality checks

```/dev/null/commands.sh#L1-2
npm run lint
npm run build
```

The `src/components/ui` directory is generated by the BambiUI CLI and is ignored by ESLint in this app. Studio-specific code lives under `src/studio`.

## Project structure

- `app/page.tsx` renders the Studio shell.
- `src/studio/components` contains Studio UI building blocks.
- `src/studio/registry` defines the component preview registry.
- `src/studio/tokens` contains editable token metadata and defaults.
- `src/studio/theme` contains theme generation, import and export helpers.
- `src/styles/bambi.css` contains BambiUI token and component CSS.

## Theme export formats

Studio supports three theme export modes:

1. CSS custom properties
2. JSON token payload
3. BambiUI preset draft

Imported files can be:

- a flat JSON token map
- a JSON object with a `tokens` field
- a CSS file containing custom property declarations

Board layouts can also be exported and imported as `bambiui-studio-canvas.json` files.
